# Renax - Car Rental Clone

A premium, modern car rental web application built with a focus on performance, aesthetics, and user experience. This project is a functional clone of the **Renax - Car Rental Template** designed by **DuruThemes**, meticulously coded from scratch to ensure high-quality standards and modern web practices.

> [!IMPORTANT]
> This project is a clone of the **Renax** design by **DuruThemes**. The entire codebase was implemented and coded by me to bring the design to life as a fully functional React application.

## ✨ Key Features

- **Modern UI/UX**: Inspired by the premium Renax template with sleek animations and a professional layout.
- **Dynamic Car Fleet**: Browse a wide range of vehicles with detailed specifications and pricing.
- **Responsive Design**: Fully optimized for all devices, from desktops to mobile phones.
- **Performance Optimized**: Built with Vite and modern React patterns for blazing-fast load times.
- **Advanced Animations**: Powered by Framer Motion for smooth transitions and interactive elements.
- **Lazy Loading**: Custom `LazyImage` components for improved performance and Lighthouse scores.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Carousels**: [Embla Carousel](https://www.embla-carousel.com/) & [Slick Carousel](https://kenwheeler.github.io/slick/)
- **Development**: [Storybook](https://storybook.js.org/) for component-driven development

## 🛠️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/iheb1919/rentCar.git
   ```

2. Navigate to the project directory:
   ```bash
   cd rentCar
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Development

- **Run Storybook**: `npm run storybook`
- **Build for Production**: `npm run build`
- **Linting**: `npm run lint`
